## Basic Summary

This is a bug fix for the "Mobile Blast" feature from [Fumi's Codex](https://github.com/Truinto/KingmakerFumi). All credits go to [Truinto](https://github.com/Truinto) for creating the original code.

This project does depend on DLL files from Fumi's Codex and Call of the Wild, so ensure that you add proper references. This project should work with 4.8 .NET & 2.0 Harmony.

The template used to modify Truinto's work was provided by [OwlcatNuGetTemplates](https://github.com/xADDBx/OwlcatNuGetTemplates/tree/main), so go visit that link if you wish to create your own custom mod.
